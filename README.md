# maus-populations-simulation
##Beschreibung
Ein selbst programmierter Mäuse Populationsberechner.
Der Nutzer gibt die anzahl der Generationen an und das Programm rechnet die Anzahl der Mäuse aus.

##Funktionen
- Eigenangabe
- Programm rechnet Gesamt und Einzel Anzahl der Population aus
